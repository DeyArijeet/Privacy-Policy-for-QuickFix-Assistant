**QuickFix Assistant** is a professional Android utility application designed to help users solve common smartphone problems through clear, educational, and step-by-step manual guidance. 

Unlike "optimizer" or "cleaner" apps, QuickFix focuses on **education and manual fixes**, guiding users to the correct Android system settings without performing any automatic or aggressive background modifications.

## üöÄ Features

- **Smart Categories:** Solutions for Battery, Storage, Performance, Apps, and Internet issues.
- **Proactive Detection:** Background monitoring that alerts users to low battery, storage space exhaustion, and suboptimal display settings.
- **Step-by-Step Guides:** Clear explanations of "Why this happens" and "How to fix it manually."
- **Direct System Integration:** One-tap buttons to open the relevant Android settings menus.
- **Privacy First:** No personal data collection, no account required, and local-only data storage.
- **Modern UI:** Built with Jetpack Compose and Material 3 for a premium, accessible user experience.
